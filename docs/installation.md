# Installation

## HACS (Recommended)

1. Open HACS in Home Assistant.
2. Click on **Integrations** and select the three dots in the top-right corner.
3. Select **Custom repositories**.
4. Add the URL of the repository: `https://github.com/FaserF/ha-mydealz` and select **Integration** as category.
5. Click **Add** and then download/install the integration.
6. Restart Home Assistant.

## Manual Installation

1. Download the latest release `.zip` file.
2. Extract and copy the `custom_components/mydealz` directory into your Home Assistant's `config/custom_components/` directory.
3. Restart Home Assistant.
