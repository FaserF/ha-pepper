# Security Policy

## Supported Versions

Only the latest version of the integration is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not** open a public issue. Instead, please report it privately:

- By opening a draft security advisory on GitHub (via the Security tab).
- Or by contacting the maintainer directly via email: `github@fabiseitz.de`.

We will investigate and attempt to address the issue promptly.
